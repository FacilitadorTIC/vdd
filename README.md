# vdd
A verbose dd command with pv and dialog.

# Example usage:
`# vdd /dev/sdX backup.iso` <br>
`# vdd test.img /dev/sdX` <br>
`# vdd /dev/sde2 /dev/sde3 "bs=128M conv=notrunc,noerror"` <br>
